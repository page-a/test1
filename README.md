# Drops - Streetwear & Hype

A brutalist sneaker drop tracker for monitoring live and upcoming releases.

## Features
- **Live Drop Tracker**: Real-time sneaker release information
- **Upcoming Releases**: See what's coming next with countdown timers
- **Resale Listings**: Access to sold-out items on the resale market
- **Dark Mode UI**: Minimalist brutalist design with Tailwind CSS

## Tech Stack
- HTML5
- Tailwind CSS
- Material Icons
